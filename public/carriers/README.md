# Carrier Logos

Place carrier logo images in this folder with the following naming convention:

- `american-amicable.png`
- `american-home-life.png`
- `americo.png`
- etc.

These images will be displayed in:
1. The carrier resource cards on the main page
2. The carrier modal headers

Recommended image specifications:
- Format: PNG or SVG
- Size: 200x200px (or similar square ratio)
- Background: Transparent preferred
